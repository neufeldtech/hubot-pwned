hubot-pwned
================

Hubot interface to the https://haveibeenpwned.com/ service.
## Installation


* Run the ```npm install``` command

```
npm install hubot-pwned --save
```

* Add the following code in your external-scripts.json file.

```
["hubot-pwned"]
```

## Usage

```
bin/hubot

has foo@bar.com been pwned?
```
